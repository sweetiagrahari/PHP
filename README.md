# PHP — Core PHP Practice

A collection of **core PHP** scripts written to practice fundamentals, including basic CRUD operations with a MySQL database.

## Contents

- `curd_read.php` — reading records from the database (CRUD basics)

## Tech Stack

- **Language:** PHP
- **Database:** MySQL

## Getting Started

1. Copy the files into your local server directory (e.g. `htdocs` for XAMPP).
2. Create the database and update the connection settings in the script.
3. Run through `http://localhost/PHP/curd_read.php`.
